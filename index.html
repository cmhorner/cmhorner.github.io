About

While coding is a relatively new enterprise for me, it satisfies a need for creativity and lifelong learning that I have carried throughout my career. I have recently (Oct 2017) completed the Full Stack Code Immersion program with Tech Talent South in Wilmington and I am currently taking their Javascript course. I had been a cook/chef for nearly 20 years and followed that with a year in Marketing and Customer Service. To say that Software Development felt like a leap of faith is a bit of an understatement, but past the challenges of learning something new I see the same joys that I saw in cooking. The camaraderie of a team environment, the steady quest for perfection, the joy of seeing something refined built out of raw ideas and the overall satisfaction of accomplishing goals.

Outside of work I'm family-oriented and I can be found most weekend spending time with my wife and pets or sharing a meal and football game with the in-laws. I'm a massive soccer fan and will happily talk your ear off about English Premier League Football and enjoy just about all types of music and movies. I feel like one of my greatest strengths is my wealth of different life experiences, making me a natural leader and team player and giving me great joy in seeing both myself and others succeed.

Twitter Clone

This Twitter Clone is a unique project in that it is one of the first times that I had the opportunity to customize an application and truly see reactive programming. It allows the user to log in, create an id and password using the Devise gem and post "tweets". Along with that, we allowed the user to upload an Avatar using Gravatar giving it personalization.

  • Client: Tech Talent South - Wilmington, NC
  • Date: September 2017
  • Technologies used: Bootstrap, HTML, CSS, Ruby on Rails, JavaScript

Marta Bus Locater

This project was one of my first introductions to APIs. It's a fun little map application where you simply enter an Atlanta address and you find out how far you are away from a bus/bus stop. We utilized the Geocoder gem for this application.

  • Client: Tech Talent South - Wilmington, NC
  • Date: October 2017
  • Technologies used: Ruby on Rails, HTML, CSS, Bootstrap, Javascript

Ecommerce Online Store

This was a really fun project. It allowed us to use the Stripe API and see a cart built and create an actual inventory. We utilized the Devise gem as well as Bootstrap to help with some functionality for the user. It led to good discussion on UX and also on tracking inventory through programs like SQL.

  • Client: Tech Talent South - Wilmington, NC
  • Date: October 2017
  • Technologies used: Ruby on Rails, Bootstrap, Javascript, SQL

Stock Trading Social App

This is a project that I began work on with a fellow classmate. We wanted to see if we could utilize some of the elements that we had learned from previous apps like the Twitter clone and the E-commerce store and apply them to another application while building the Rails project from scratch. While it was a lot of work, it gave me a greater understanding for how Rails functions and the relationship between each piece of the application. The app itself is designed to so that you can log in, create a user, pull any stock number (using an API that retrieves opening and closing stock numbers) and store up to ten at a time on your page. You can also maintain a friend network and watch their stocks and they yours, choosing to follow any that are interesting.

  • Client: Tech Talent South - Wilmington, NC
  • Date: October 2017
  • Technologies used: Ruby on Rails, HTML, Bootstrap
